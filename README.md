# EduLearn - Premium E-Learning Platform

A modern, responsive e-learning platform built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Modern UI**: Glassmorphism design with a dark theme.
- **Course Discovery**: Browse and search through various tech courses.
- **User Dashboard**: Track your enrolled courses and progress.
- **Responsive**: Works on desktop, tablet, and mobile.

## How to Run Locally
1. Clone the repository.
2. Open `index.html` in your browser.
3. (Optional) Run a local server:
   ```bash
   npx serve .
   ```

## Deployment
This project is ready to be hosted on **GitHub Pages**. Simply push the code to a repository and enable GitHub Pages in the settings.
